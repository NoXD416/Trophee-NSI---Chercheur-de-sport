La course à pied est une discipline sportive de l'athlétisme. l existe cinq types de courses, les courses courtes : le sprint, les courses mi-longues : le demi-fond, et les courses longues, le fond. Les deux derniers type sont les courses d'obstacles et la marche athlétique.
Ses techniques sont:

Une bonne technique de respiration

Un placement optimal du pied en courant

Cette activité permet d'améliorer les conditions physiques des organes indispensables à la vie comme le cœur, le poumon, les artères ou encore le foie.
